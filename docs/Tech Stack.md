# Tech Stack

## Backend
- **Language**: Python 3.10+
- **Framework**: FastAPI 0.104+
- **Database**: PostgreSQL (NeonDB)
- **ORM**: SQLAlchemy 2.0+
- **Validation**: Pydantic 2.0+
- **ML Frameworks**:
  - PyTorch 2.1+
  - TorchVision 0.16+
  - TensorFlow 2.15+
  - Keras 3.0+
- **Optimization Libraries**:
  - torch.quantization (PyTorch quantization)
  - tensorflow_model_optimization (TensorFlow optimization)
  - torch.nn.utils.prune (PyTorch pruning)
- **WebSocket**: Built-in FastAPI WebSocket support
- **File Upload**: python-multipart
- **Environment**: python-dotenv
- **Testing**: pytest, pytest-asyncio

## Frontend
- **Language**: TypeScript 5.0+
- **Framework**: React 18+
- **Build Tool**: Vite
- **HTTP Client**: Axios 1.6+
- **WebSocket**: Native WebSocket API
- **Routing**: React Router 6+
- **Styling**: Tailwind CSS 3.0+
- **Charts**: Recharts 2.10+
- **State Management**: React Context API (built-in)
- **Forms**: React Hook Form (optional for complex forms)

## DevOps
- **Containerization**: Docker, Docker Compose
- **Database**: PostgreSQL 15+ via NeonDB
- **Environment Management**: .env files

## Development Tools
- **Backend Linting**: ruff, black
- **Frontend Linting**: ESLint, Prettier
- **Type Checking**: mypy (Python), TypeScript compiler
- **Testing**: pytest (backend), Vitest/Jest (frontend)
